## How I work

- **Opinionated over configurable.** Small tools that do one thing well
  beat Swiss Army knives. If a setting doesn't change what most users do,
  it shouldn't be a setting.
- **No black boxes.** Everything a tool does should be inspectable. If
  it cleans, list what it cleans. If it tracks, show the numbers.
- **Performance is a feature.** Not a phase after shipping. Freeze =
  bug, not "we'll optimize later".
- **Copy matters.** The difference between a trustworthy tool and a
  suspicious one is usually three sentences of honest writing.
