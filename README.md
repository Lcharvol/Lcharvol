## How I work

- **Opinionated over configurable.** If a setting doesn't change what
  most people do, it shouldn't be a setting.
- **No black boxes.** Everything a tool does should be inspectable. If
  it cleans, it lists what it cleaned. If it tracks, it shows the numbers.
- **Performance is a feature.** Not a phase after shipping. A freeze is
  a bug, not "we'll optimize later".
- **Copy matters.** The difference between a trustworthy tool and a
  suspicious one is usually three sentences of honest writing.
